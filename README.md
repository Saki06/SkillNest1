# SkillNest

SkillNest is a full-stack learning and professional networking platform where users can build profiles, publish posts, showcase projects, upload documents, search members/posts, and chat in real time.

## Features
- User registration and login
- Google OAuth login
- JWT-based authentication
- Profile management (about, skills, documents, showcases, posts)
- Post, like, and comment workflows
- Advanced search with filters and autocomplete
- Real-time messaging with WebSocket (STOMP + SockJS)
- Notification support

## Tech Stack
**Frontend**
- React 19, Vite 6
- Tailwind CSS
- Axios, React Router
- SockJS + STOMP client

**Backend**
- Java 21
- Spring Boot 3.5 (snapshot)
- Spring Security
- Spring Data MongoDB
- Spring WebSocket
- JWT (jjwt)

**Database**
- MongoDB
